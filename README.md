# QClaw Skip Invite

跳过 [QClaw](https://claw.guanjia.qq.com/) 应用的邀请码验证，去除启动时的邀请码弹窗。

## 支持

- QClaw v0.1.9+（macOS / Windows）

## 前置要求

- Node.js >= 22

## 使用

```bash
npx qclaw-skip-invite
```

如果 QClaw 正在运行，工具会自动关闭并在完成后重启。该命令可重复执行，已打过补丁会自动跳过。

## 还原

重新安装 QClaw 即可还原。
